Vertica的这些事（一）——VSQL常用命令

发表于 2018-09-12

Linux环境下，一般都是使用vsql客户端连接vertica来执行命令的，下面就介绍一下vsql常用的命令有哪些。

导出数据

默认是以‘|’分割的，指定分隔符如下：

切换路径 ：\cd

列出多有表：\d
列出多有函数：\df
列出所有projection：\dj
列出所有的schema：\dn
列出所有的序列：\ds
列出所有的系统字典表：\dS
列出所有支持的类型：\dT
列出所有的视图：\dv
编辑sql：\e

执行缓存的sql：\g
输出HTML格式的结果：\H

查看当前缓存buffer里的内容：\p

修改密码：\password [ USER ]

情况当前buffer：\r

历史命令查看保存为file：\s [ FILE ]

查看所有表的权限：\dp 或者\z

copy:

直接copyHDFS的文件到vertica
